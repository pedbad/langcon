{% icon 'rocket' class_='h-6 w-6 text-foreground/80' %} {% c card/title %}Reuse this Base for a New Project{% endc %}
{% c card/description %} Follow these steps after downloading the ZIP to create a new project (e.g. LangCon, Pace, Ice). {% endc %} {% endc %} {% c card/content %}
  1. {% icon 'folder' class_='mt-0.5 h-4 w-4 shrink-0' %} Rename the root folder (e.g. langcen_base → langcon). Then open it in your editor.
  2. {% icon 'replace' class_='mt-0.5 h-4 w-4 shrink-0' %} Search & replace branding: “LangCen / LangCen Base” → your name; update titles, UNFOLD settings, and meta tags.
  3. {% icon 'package' class_='mt-0.5 h-4 w-4 shrink-0' %} Install deps: pip install -r requirements*.txt (and npm install if using Tailwind build/watch).
  4. {% icon 'settings' class_='mt-0.5 h-4 w-4 shrink-0' %} Project identity: update UNFOLD titles, DEFAULT_FROM_EMAIL, and admin site headers if needed.
  5. {% icon 'db' class_='mt-0.5 h-4 w-4 shrink-0' %} Migrate & create a superuser: python src/manage.py migrate then createsuperuser.
  6. {% icon 'brush' class_='mt-0.5 h-4 w-4 shrink-0' %} Swap logos/icons in src/core/static/core/img/, update favicons and Open Graph image.
  7. {% icon 'play' class_='mt-0.5 h-4 w-4 shrink-0' %} Run: python src/manage.py runserver. Verify dark/light theme, favicon, login, and admin titles.
